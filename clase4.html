Clase 4: Fórmulas matemáticas y bibliografía

1. Escritura de fórmulas matemáticas (1.5h)

1.1 Modo inline y modo display

Inline: Fórmulas dentro del texto, usando $...$ o \(...\).

La energía $E = mc^2$ es fundamental en física moderna.
La integral \(\int_a^b f(x)dx\) representa el área bajo la curva.

Display: Fórmulas centradas y destacadas, con \[...\] o equation*.

La ecuación de onda es:
\[
\nabla^2 \psi = \frac{1}{v^2} \frac{\partial^2 \psi}{\partial t^2}
\]

1.2 Símbolos matemáticos comunes

Letras griegas: \alpha, \beta, \Gamma, \Omega.

Operadores: \times, \div, \pm, \sum, \int.

Fracciones y raíces:

\(\frac{a}{b}\), \(\sqrt{x}\), \(\sqrt[n]{x}\).
\[
\sum_{i=1}^n \frac{\sqrt{x_i}}{2} \times \alpha
\]

1.3 Ecuaciones numeradas y referencias

Entorno equation para numeración automática:

\begin{equation}
    \label{eq:onda}
    \frac{\partial^2 u}{\partial t^2} = c^2 \nabla^2 u
\end{equation}
Como se ve en la ecuación \eqref{eq:onda}, la velocidad es crítica.

Consultar la Comprehensive LaTeX Symbol list y Detexify para encontrar símbolos matemáticos.

2. Introducción a BibTeX y JabRef (1.5h)

2.1 Crear una base de datos bibliográfica

Estructura de un archivo .bib:

@book{einstein1916relativity,
    title     = {Relativity: The Special and General Theory},
    author    = {Einstein, Albert},
    year      = {1916},
    publisher = {Methuen & Co.}
}

2.2 Uso de JabRef

Pasos básicos:

  1. Crear nueva entrada (Ctrl + N).
  2. Seleccionar tipo (libro, artículo, etc.).
  3. Rellenar campos: autor, título, año, DOI (si existe).
  4. Guardar como referencias.bib.
  5. Búsquedas en bases de dados (ej. PubMed)
  6. Inserción automática de citas en el documento

2.3 Citas y bibliografía en LaTeX

Preámbulo:

\usepackage{natbib}  % Para estilos de citas
\bibliographystyle{plainnat}  % Estilo APA, IEEE, etc.

Citar en el texto:

Según \citet{einstein1916relativity}, la teoría...
Estudios recientes \citep{hawking1973large} confirman...

Generar la bibliografía:

\bibliography{referencias}  % Sin extensión .bib

2.4 Compilación con BibTeX

Flujo de comandos (en orden):

  1. pdflatex documento.tex
  2. bibtex documento.aux
  3. pdflatex documento.tex (2 veces).

3. Ejercicios prácticos (1h)

Ejercicio 1: Fórmulas matemáticas

Crea un documento LaTeX que incluya:

  • Una ecuación numerada con integral y fracciones:
    • \begin{equation}
    \label{eq:integral}
    \int_0^1 \frac{x^2}{\sqrt{1 - x^2}} dx = \frac{\pi}{8}
\end{equation}
  • Una fórmula en línea con una matriz:
    • La matriz identidad: \(
I = \begin{pmatrix}
  1 & 0 \\
  0 & 1
\end{pmatrix}
\)

Ejercicio 2: Bibliografía con JabRef

  1. Añade 2 entradas a referencias.bib usando JabRef (un libro y un artículo).
  2. Cita ambas fuentes en el texto.
  3. Compila el documento y verifica que la bibliografía se genera correctamente.

Errores comunes y soluciones

  • ¡La bibliografía no aparece!:
    • Asegúrate de ejecutar bibtex y compilar 2 veces con pdflatex.
  • Citas con ?:
    • Verifica que la clave en \cite coincida con la entrada en .bib.
  • JabRef no guarda campos:
    • Usa Ctrl + S después de editar cada entrada.

Recursos adicionales

  • Lista completa de símbolos matemáticos: Detexify.
  • Estilos bibliográficos: Overleaf - BibTeX.

Otros estilos de bibliografía

Además de natbib, LaTeX ofrece múltiples paquetes y estilos para gestionar citas y bibliografía. Aquí tienes una lista de los más relevantes, con ejemplos y casos de uso:

1. biblatex

Ventaja: Es el paquete más moderno y flexible, compatible con BibLaTeX (sucesor de BibTeX).

Estilos populares:

  • style=authoryear: Citas como (Autor, Año).
\usepackage[style=authoryear]{biblatex}
\addbibresource{referencias.bib} % Cargar base de datos
\cite{einstein1916} → (Einstein, 1916)
  • style=numeric: Citas numéricas [1].
  • style=apa: Estilo APA (requiere biblatex-apa).

Ejemplo completo:

\usepackage[backend=biber, style=authoryear]{biblatex}
...
\printbibliography % Generar bibliografía

2. Estilos clásicos de BibTeX

Funcionan con \bibliographystyle{...} (sin paquetes adicionales):

  • plain: Citas numéricas y autores en orden alfabético.
  • unsrt: Como plain, pero ordenadas por aparición en el texto.
  • alpha: Citas con abreviaturas (ej: [Ein16]).
  • abbrv: Versión abreviada de plain (nombres cortos).

Ejemplo:

\bibliographystyle{plain}
\bibliography{referencias}

3. apacite (APA)

Para trabajos en ciencias sociales:

\usepackage{apacite}
\bibliographystyle{apacite}
...
\citeA{einstein1916} → Einstein (1916)
\cite{hawking1973} → (Hawking, 1973)

Recomendaciones según disciplina

  • Ciencias naturales: natbib o biblatex (estilo authoryear).
  • Ingeniería: IEEEtran.
  • Ciencias Sociales: apacite o biblatex-apa.
  • Humanidades: chicago o oxford.

Nota: Si usas biblatex, compila con biber en lugar de bibtex (en TeXStudio: Herramientas > Comandos > Biber).
