Quickstart: Primeros pasos en Overleaf

¿Por qué Overleaf?

Overleaf es una plataforma cloud para trabajar con LaTeX sin necesidad de instalar nada en tu ordenador. Todo lo que necesitas es un navegador web y una conexión a internet. Colaboración en tiempo real, control de versiones integrado, y compilación automática.

1. Crear tu cuenta en Overleaf

  1. Ve a https://www.overleaf.com
  2. Haz clic en "Sign Up" (esquina superior derecha)
  3. Completa el formulario con:
    • Tu dirección de correo electrónico
    • Nombre y apellidos
    • Contraseña segura
  4. Verifica tu correo electrónico (revisa spam si es necesario)
  5. ¡Tu cuenta está lista!

2. Acceder al portal institucional (Campus License)

Nota: Si tu universidad tiene licencia de campus, puedes acceder con credenciales institucionales:

  1. Ve a https://www.overleaf.com
  2. En la página de login, busca la opción "Institution Login"
  3. Busca tu universidad en la lista
  4. Inicia sesión con tus credenciales de la universidad
  5. Tendrás acceso a todas las características premium de forma gratuita

3. Crear tu primer proyecto

  1. Una vez logueado, haz clic en "New Project"
  2. Elige una opción:
    • "Blank Project": Para empezar desde cero
    • "Example Project": Para ver ejemplos de LaTeX
    • "Upload Project": Si tienes archivos locales
  3. Dale un nombre a tu proyecto (ej: "Mi primer documento")
  4. ¡Listo! Accederás al editor de Overleaf

4. Familiarizarte con la interfaz

La pantalla de Overleaf tiene tres secciones principales:

  • Panel izquierdo: Estructura de archivos de tu proyecto
  • Panel central: Editor de código LaTeX
  • Panel derecho: Vista previa del documento compilado

Prueba escribiendo código LaTeX en el editor y verás el resultado en tiempo real en la preview.

5. Colaboración en tiempo real

  1. Abre tu proyecto
  2. Haz clic en "Share" (botón azul, arriba a la derecha)
  3. Copia el enlace o invita a compañeros por email
  4. Elige el nivel de acceso:
    • Can edit: Pueden modificar el proyecto
    • Can view only: Solo lectura
  5. ¡Ahora otros pueden editar el documento contigo en tiempo real!

6. Control de versiones integrado

Overleaf tiene Git integrado de forma sencilla:

  1. En tu proyecto, haz clic en el menú (≡) a la izquierda
  2. Selecciona "History"
  3. Verás todos los cambios realizados en el documento
  4. Puedes deshacer cambios haciendo clic en cualquier versión anterior

Para usuarios avanzados: También puedes clonar proyectos de Overleaf usando Git desde la terminal. Consulta Clase 1 para más detalles.

7. Síntesis de herramientas Overleaf vs herramientas locales

Función Herramienta Local Overleaf
Editar LaTeX TeXstudio Editor web (sin instalación)
Compilar PDF pdflatex desde terminal Automático en tiempo real
Control de versiones Git desde terminal History integrado (+ Git opcional)
Bibliografía JabRef (local) Zotero + Overleaf (cloud)
Colaboración Compartir archivos (manual) Tiempo real (integrado)
