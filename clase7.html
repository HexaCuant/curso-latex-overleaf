Clase 7: Temas especiales

Creación de una carta con membrete en LaTeX

Usando la clase letter:

Ejemplo básico de una carta formal en LaTeX:

\documentclass[a4paper,12pt]{letter}
\usepackage[utf8]{inputenc}
\usepackage[spanish]{babel}
\usepackage[T1]{fontenc}

% Definición de la dirección del remitente
\address{Calle Falsa 123\\ Ciudad de México, CP 01000}

% Definición de la firma del remitente
\signature{Juan Pérez}

% Opcionalmente, se puede definir la fecha
\date{\today}

\begin{document}

% Inicio del entorno de la carta
\begin{letter}{Lic. María López\\ Departamento de Comunicaciones\\ 
Empresa XYZ\\ Calle Real 456\\ Otra Ciudad, CP 02000}

% Declaración de apertura
\opening{Estimada Lic. López:}

Me permito escribirle la presente carta con el propósito de informarle 
sobre la nueva estrategia de comunicación que implementaremos en la empresa. 
La iniciativa está orientada a mejorar la interacción con nuestros clientes 
mediante el uso de medios digitales y herramientas innovadoras.

Agradezco de antemano su atención y quedo a su disposición para ampliar 
cualquier información adicional que requiera.

% Declaración de cierre
\closing{Atentamente,}

% Inclusión de una posdata
\ps{PD: Adjunto el documento con el plan de acción para su revisión.}

% Inclusión de documentos adjuntos
\encl{Plan de Acción\\ Cronograma de Actividades}

% Inclusión de destinatarios en copia
\cc{Dr. Carlos Gómez\\ Ing. Marta Díaz}

\end{letter}

\end{document}

Elementos principales:

  • Clase letter específica para cartas
  • Dirección del remitente con \address
  • Firma con \signature
  • Fecha automática con \date{\today}
  • Entorno letter para el contenido
  • Elementos opcionales: PS, adjuntos y copias

Ejemplo de código para insertar logos en las esquinas superiores:

\documentclass[a4paper]{letter}
  \usepackage[margin=1in]{geometry}       % Define márgenes estándar
  \usepackage{eso-pic}                    % Permite añadir imágenes en posiciones absolutas
  \usepackage{graphicx}                   % Necesario para la inclusión de gráficos
  
  \newsavebox{\mylogo}
  \savebox{\mylogo}{%
    \parbox{4cm}{%
      \centering%
      \includegraphics[width=2cm]{logo_ib}\\
      {\scriptsize\textbf{\emph{Instituto de Biotecnología}\\ Universidad de Granada}}%
    }%
  }
  
  
  \newcommand{\AtPageUpperRight}[1]{%
    \AtPageUpperLeft{%
      \makebox[1.2\textwidth][r]{#1}%
    }%
  }
  
  
  % Comando para agregar un logo en la esquina superior izquierda
  \newcommand{\AddLeftLogo}{
    \AddToShipoutPictureBG*{%
      \AtPageUpperLeft{%
        \hspace{1.5cm}%
        \raisebox{-3cm}{\centering\includegraphics[height=2cm]{escudougr.png}}%
      }%
    }%
  }
  
  % Comando para agregar un logo en la esquina superior derecha
  \newcommand{\AddRightLogo}{
    \AddToShipoutPictureBG*{%
      \AtPageUpperRight{%
        \raisebox{-3cm}{\usebox{\mylogo}}%
      }%
    }%
  }
  
  %\includegraphics[height=2cm]{logo_ib.png}
  
  \signature{Fulanito de Tal}
  
  \begin{document}
    % Añadimos ambos logos a cada página (o solo a la primera según la implementación)
    \AddLeftLogo
    \AddRightLogo
    
    \begin{letter}{Nombre del destinatario \\ Dirección \\ Ciudad}
      \opening{Estimado/a,}
      
      Contenido de la carta. Aquí se pueden incluir párrafos, listas y cualquier otro elemento de LaTeX.
      
      \closing{Atentamente,}
    \end{letter}
  \end{document}

Este ejemplo muestra cómo utilizar eso-pic para incorporar imágenes fuera de los márgenes, ajustando la posición horizontal (con \hspace) y vertical (con \raisebox) para evitar que los logos queden demasiado pegados a los bordes.

Usando la clase article:

Utilizar article en lugar de letter es viable y puede ser una opción atractiva si se necesita incorporar elementos que no están disponibles en la clase letter o si se prefiere una mayor flexibilidad en el diseño, a costa de implementar algunos detalles manualmente para recrear la estructura estándar de una carta.

Paso 1: Configuración inicial del documento

Código:

\documentclass[a4paper,12pt]{article}
    \usepackage[utf8]{inputenc}
    \usepackage[spanish]{babel}

Explicación:

  • \documentclass[a4paper,12pt]{article}: Define el tipo de documento (artículo) con tamaño de hoja A4 y fuente de 12pt.
  • inputenc y babel: Permiten escribir en español con codificación UTF-8 (acentos, ñ, etc.).

Paso 2: Paquetes esenciales y márgenes

Código:

\usepackage{graphicx}
    \usepackage[top=4cm,headheight=4cm]{geometry}
    \usepackage{fancyhdr}
    \usepackage{setspace}

Explicación:

  • graphicx: Para insertar imágenes (logo, firma).
  • geometry: Configura márgenes. top=4cm deja espacio para el membrete.
  • fancyhdr: Personaliza encabezados/pies de página.
  • setspace: Controla el interlineado.

Paso 3: Definir el membrete en el encabezado

Código:

\newsavebox{\mylogo}
    \newsavebox{\mylogo}
    \savebox{\mylogo}{\includegraphics[width=2cm]{escudougr}}
    
    \newsavebox{\logoderecha}
    \savebox{\logoderecha}{\includegraphics[width=2cm]{logo_ib}}
    
    \lhead{

      \begin{minipage}[t]{4cm}
      \vspace{1cm}\hspace{-2cm}\parbox[t]{4cm}{
        \begin{center}
          \usebox{\mylogo} \\
          \tiny{\textbf{Facultad de Ciencias\\Universidad de Granada}}
        \end{center}
        }
      \end{minipage}
    }
    
    
    \rhead{
    \begin{minipage}[t]{4cm}
    \vspace{1cm}\hspace*{2cm}\parbox[t]{4cm}{
    \begin{center}
    \usebox{\logoderecha} \\
    \tiny{\textbf{\emph{Intituto de Biotecnología}}}
    \end{center}
    }
    \end{minipage}
    }
    
    \renewcommand{\headrulewidth}{0pt}

Clase: Creación de Pósters en LaTeX

Paso 1: Configuración inicial del documento

Objetivo: Definir la clase del documento y paquetes esenciales.

\documentclass[a0,portrait]{a0poster}
    \usepackage[utf8]{inputenc}
    \usepackage[USenglish]{babel}
  • Explicación:
    • a0poster: Clase específica para pósteres tamaño A0 en orientación vertical (portrait).
    • inputenc y babel: Manejan codificación UTF-8 y traducciones al inglés estadounidense.

Paso 2: Paquetes adicionales y personalizaciones

Objetivo: Añadir funcionalidades gráficas, de texto y estructura.

\usepackage{marvosym, multirow, rotating, setspace, multicol, color, type1cm, colortbl}
    \usepackage{shapepar, pstricks, eso-pic}
  • Paquetes clave:
    • multicol: Divide el contenido en columnas.
    • color y colortbl: Permiten usar colores en texto y tablas.
    • eso-pic: Para insertar imágenes de fondo.
    • pstricks: Herramientas gráficas avanzadas.

Paso 3: Configuración del fondo del póster

Objetivo: Insertar una imagen como fondo.

\newcommand\BackgroundPicture{%
     \put(0,0){%
     \parbox[b][\paperheight]{\paperwidth}{%
     \vfill
     \centering
     \includegraphics[width=\paperwidth,height=\paperheight,keepaspectratio]{esc-ugr-claro}%
     \vfill
     }}}
    \AddToShipoutPicture{\BackgroundPicture}
  • Funcionamiento:
    • \BackgroundPicture centra la imagen esc-ugr-claro en todo el póster.
    • \AddToShipoutPicture aplica este fondo a todas las páginas.

Paso 4: Definición de comandos personalizados

Objetivo: Simplificar la escritura de títulos, secciones y texto.

4.1 Título principal
\newcommand{\titulo}[4]{
     \begin{minipage}[c]{10cm}
     \includegraphics[width=10cm]{#1}
     \end{minipage}
     \hfill
     \begin{minipage}[c]{65cm}
     \fontsize{2.2cm}{2.6cm}\selectfont 
     \centering \textbf{\textcolor[rgb]{.5,0,0}{#2}}
     \begin{center}
     \fontsize{1.8cm}{1.8cm}\selectfont
     \textbf{\textcolor[rgb]{0,0,.5}{#3}}
     \vspace{.5cm}
     \textbf{\textcolor[rgb]{0,.5,0}{#4}}
     \end{center}
     \end{minipage}
     \vspace{1cm}
     \rule{\textwidth}{.2cm}
    }

Uso: \titulo{logo.png}{Título}{Autores}{Institución}

  • #1: Ruta del logo.
  • #2, #3, #4: Título, autores e institución con colores predefinidos.

Paso 5: Estructura del contenido

Objetivo: Organizar el texto en columnas y añadir figuras.

5.1 Columnas
\begin{multicols*}{2}
    ...
    \end{multicols*}

Divide el contenido en 2 columnas. El asterisco (*) evita equilibrar alturas automáticamente.

Paso 6: Ajustes finales

Objetivo: Personalizar espaciados y reglas visuales.

\setlength{\parindent}{3cm}
    \setlength{\parskip}{.5em}
    \renewcommand{\columnseprule}{3pt}
    \setlength{\columnsep}{3cm}

Paso 7: Compilación

Recomendaciones:

  • Usar un compilador que soporte pstricks (ej: XeLaTeX o LuaLaTeX).
  • Verificar rutas de imágenes y fuentes.

Resumen de flujo de trabajo:

  1. Configurar clase y paquetes.
  2. Definir imagen de fondo.
  3. Crear comandos personalizados para títulos/secciones.
  4. Usar multicols* para dividir en columnas.
  5. Insertar figuras con minipage y framed.
  6. Ajustar espaciados y reglas visuales.
  7. Compilar y revisar formato final.

Otras opciones

beamerposter:
Se basa en la clase beamer habitual para presentaciones, pero incorpora estilos y formatos pensados para posters de gran tamaño. Su estructura te permite definir bloques de contenido y distribuir la información en columnas. Un ejemplo básico es:

\documentclass{beamer}
  \usepackage[orientation=portrait,size=a0,scale=1.4]{beamerposter}
  \usetheme{Warsaw}
  \usepackage[utf8]{inputenc}
  \usepackage[spanish]{babel}
  
  \title{Título del Póster}
  \author{Nombre del Autor}
  \institute{Institución}
  \date{Fecha}
  
  \begin{document}
  \begin{frame}[plain]
    \maketitle
    \block{Introducción}{
      Aquí se introduce el tema del póster.
    }
    \block{Métodos}{
      Descripción de la metodología.
    }
    \block{Resultados}{
      Presentación de datos y gráficos.
    }
  \end{frame}
  \end{document}

Este método es especialmente útil si ya estás familiarizado con beamer y deseas aprovechar sus comandos para estructurar el contenido de forma sencilla.

tikzposter:
Si buscas una mayor flexibilidad en la disposición gráfica, la clase tikzposter te permite diseñar pósters con estructuras muy personalizables usando comandos del paquete TikZ. Un ejemplo básico sería:

\documentclass[25pt, a0paper, portrait]{tikzposter}
  \title{Título del Póster}
  \author{Nombre del Autor}
  \date{Fecha}
  \usetheme{Basic}
  
  \begin{document}
  \maketitle
  \block{Sección 1}{
    Contenido de la sección.
  }
  \block{Sección 2}{
    Más información.
  }
  \end{document}

Esta opción es ideal para diseños creativos, donde la disposición de elementos y el uso de colores personalizables son primordiales.

Diferencias Clave

Característica beamerposter tikzposter
Enfoque Basado en la estructura de beamer Basado en TikZ, permitiendo diseños gráficos complejos
Facilidad de uso Adecuado para quienes ya usan beamer Mayor flexibilidad pero requiere familiaridad con TikZ
Estructuración Usa bloques y columnas prediseñados Permite layouts completamente personalizados

Recomendaciones Adicionales

  • Plantillas predefinidas:
    Plataformas como Overleaf ofrecen numerosos ejemplos y plantillas que te ayudan a iniciar tu proyecto sin tener que diseñar un póster desde cero. Puedes explorar la galería de plantillas de pósteres para encontrar una que se ajuste a las especificaciones de tu congreso.
  • Adaptación a Normas del Congreso:
    Algunos congresos proporcionan sus propias plantillas (por ejemplo, "Plantilla-CASEIB-2024") que debes seguir para el envío de trabajos.

Creación de presentaciones en LaTeX

Estructura Básica de una Presentación Beamer

Estructura básica del código:

\documentclass[11pt]{beamer}
    \usepackage[utf8]{inputenc}
    \usepackage[T1]{fontenc}
    \usepackage{lmodern}
    \usetheme{default}
    \begin{document}
      \author{Título de mi tesis}
      \title{yo mismo}
      %\subtitle{}
      %\logo{}
      %\institute{}
      %\date{}
      %\subject{}
      %\setbeamercovered{transparent}
      %\setbeamertemplate{navigation symbols}{}
      \begin{frame}[plain]
        \maketitle
      \end{frame}
      
      \begin{frame}
        \frametitle{Titulo de la diapositiva}
        Contenido
      \end{frame}
    \end{document}

Ejercicios: Descomentar los comandos para ver su uso y cambiar de tema.

Personalización de Colores y Fuentes

Beamer permite personalizar los colores y fuentes de tu presentación. Puedes cambiar el tema de colores con \usecolortheme{nombre} y el tema de fuentes con \usefonttheme{nombre}.

\documentclass{beamer}
  \usecolortheme{seahorse}
  \usefonttheme{serif}

  \begin{document}
    \begin{frame}
      \frametitle{Slide con Tema de Color y Fuente}
      Este slide utiliza el tema de color "seahorse" y fuente "serif".
    \end{frame}
  \end{document}

Creación de Diapositivas con Elementos Visuales

1. Columnas

Puedes dividir un slide en columnas para presentar información de manera organizada:

\begin{frame}
    \frametitle{Columnas}
    \begin{columns}
      \begin{column}{0.5\linewidth}
        Contenido en la columna izquierda.
      \end{column}
      \begin{column}{0.5\linewidth}
        Contenido en la columna derecha.
      \end{column}
    \end{columns}
  \end{frame}

2. Listas y Enumeraciones

Las listas y enumeraciones funcionan igual que en LaTeX estándar:

\begin{frame}
    \frametitle{Listas y Enumeraciones}
    \begin{itemize}
      \item Elemento 1
      \item Elemento 2
    \end{itemize}
    
    \begin{enumerate}
      \item Elemento enumerado 1
      \item Elemento enumerado 2
    \end{enumerate}
  \end{frame}

3. Imágenes

Incluye imágenes en tus diapositivas con \includegraphics:

\begin{frame}
    \frametitle{Imágenes}
    \includegraphics[width=0.5\linewidth]{nombre_imagen.jpg}
  \end{frame}

Transiciones y Animaciones

1. Transiciones entre Diapositivas

Puedes establecer transiciones entre diapositivas con \transduration y \transwipe:

\begin{frame}
    \frametitle{Transición entre Diapositivas}
    \transduration{0.5}
    \transwipe
    Contenido del slide.
  \end{frame}

Tipos de Transiciones

Beamer ofrece varios tipos de transiciones que puedes usar:

  • \transfade: Desvanecimiento.
  • \transdissolve: Disolución.
  • \transglitter: Brillantez.
  • \transwipe: Barrido.
  • \transsplit: División.
  • \transboxout: Caja que sale.
  • \transboxin: Caja que entra.

Puedes probar diferentes tipos de transiciones para ver cuál se adapta mejor a tus necesidades.

2. Animaciones con Overlays

Utiliza \pause o <+-> para crear animaciones dentro de una diapositiva:

\begin{frame}
    \frametitle{Animación con Overlays}
    \begin{itemize}
      \item Elemento 1
      \pause
      \item Elemento 2
      \pause
      \item Elemento 3
    \end{itemize}
  \end{frame}

Creación de un Índice

Puedes crear un índice para tu presentación con \tableofcontents.

\begin{frame}
    \frametitle{Índice}
    \tableofcontents
  \end{frame}
Añadir comandos de seccionado para que aparezcan en el índice.

Ejemplo Completo

Aquí tienes un ejemplo completo que combina varios elementos:

\documentclass{beamer}
  \usetheme{Madrid}
  \usecolortheme{seahorse}
  \usefonttheme{serif}

  \title{Mi Presentación Completa}
  \author{Tu Nombre}
  \institute{Tu Institución}
  \date{\today}

  \begin{document}
    \begin{frame}
      \titlepage
    \end{frame}
    
    \begin{frame}
      \frametitle{Índice}
      \tableofcontents
    \end{frame}
    
    \section{Introducción}
    \begin{frame}
      \frametitle{Introducción}
      Este es el contenido de la introducción.
    \end{frame}
    
    \section{Sección 1}
    \begin{frame}
      \frametitle{Sección 1}
      \begin{columns}
        \begin{column}{0.5\linewidth}
          Contenido en la columna izquierda.
        \end{column}
        \begin{column}{0.5\linewidth}
          Contenido en la columna derecha.
        \end{column}
      \end{columns}
    \end{frame}
    
    \section{Sección 2}
    \begin{frame}
      \frametitle{Sección 2}
      \begin{itemize}
        \item Elemento 1
        \pause
        \item Elemento 2
        \pause
        \item Elemento 3
      \end{itemize}
    \end{frame}
    
    \begin{frame}
      \frametitle{Imágenes}
      \includegraphics[width=0.5\linewidth]{nombre_imagen.jpg}
    \end{frame}
  \end{document}

Tipos de Cajas en Beamer

Beamer ofrece varios tipos de cajas que puedes utilizar para resaltar información en tus diapositivas. Estos incluyen:

  • Bloque Estándar (Standard Block)
  • Bloque de Alerta (Alert Block)
  • Bloque de Ejemplo (Example Block)
  • Bloques Matemáticos (Theorem, Definition, Proof, etc.)

Ejemplo de Código

Aquí tienes un ejemplo que muestra cómo usar estos bloques:

\documentclass{beamer}
  \usetheme{Madrid}

  \begin{document}

  \begin{frame}
    \frametitle{Tipos de Bloques}
    
    % Bloque Estándar
    \begin{block}{Bloque Estándar}
      Este es un bloque estándar.
    \end{block}
    
    % Bloque de Alerta
    \begin{alertblock}{Bloque de Alerta}
      Este es un bloque de alerta.
    \end{alertblock}
    
    % Bloque de Ejemplo
    \begin{exampleblock}{Bloque de Ejemplo}
      Este es un bloque de ejemplo.
    \end{exampleblock}
    
    % Bloque Matemático: Teorema
    \begin{theorem}
      Este es un teorema.
    \end{theorem}
    
    % Bloque Matemático: Definición
    \begin{definition}
      Esta es una definición.
    \end{definition}
    
    % Bloque Matemático: Prueba
    \begin{proof}
      Esta es una prueba.
    \end{proof}
  \end{frame}

  \end{document}

Personalización de Bloques

Puedes personalizar el estilo de los bloques utilizando los siguientes comandos:

Rodear los bloques con sombra y bordes redondeados:

\setbeamertemplate{blocks}[rounded][shadow=true]

Cambiar los colores de los bloques:

Cada tipo de bloque tiene su propi definición y unos pueden estar basados en otros, por lo que la forma de cambiar los colores depende de como esté definido cada tipo de bloque

Utiliza \setbeamercolor para cambiar los colores de los bloques. Por ejemplo:

\setbeamercolor{block title}{fg=white,bg=blue}
  \setbeamercolor{block body}{fg=black,bg=white}

para alertblock

\setbeamercolor{block title alerted}{fg=white,bg=red}
  \setbeamercolor{block body alerted}{bg=orange!25}

para exampleblock

\setbeamercolor{block title example}{fg=white, bg=teal}
  \setbeamercolor{block body example}{bg=teal!25}

Para theorem y proof hay que añadir los colores a la sección de begin

\addtobeamertemplate{theorem begin}{%
    \setbeamercolor{block title}{fg=white,bg=gray}%
    \setbeamercolor{block body}{fg=red,bg=white}%
    }{}
    
      \addtobeamertemplate{proof begin}{%
    \setbeamercolor{block title}{fg=white,bg=red}%
    \setbeamercolor{block body}{fg=green,bg=white}%
    }{}

Estas personalizaciones te permiten adaptar los bloques a tu estilo de presentación.

Otro tipo de cajas

Los entornos «beamerboxesrounded» permiten crear recuadros con esquinas redondeadas y ofrecen algunas opciones de configuración que se pasan como claves en un formato key=value dentro de los corchetes. En concreto, las opciones conocidas y más utilizadas son:

  • upper: Define el color de fondo de la parte superior (la "cabeza") del recuadro, que suele contener el título.
  • lower: Especifica el color de fondo de la parte inferior (el cuerpo) del recuadro, donde se muestra el contenido.
  • shadow: Al establecer shadow=true se añade una sombra al recuadro, lo que aporta un efecto visual de profundidad.

Por ejemplo, se puede usar de la siguiente manera:

\setbeamercolor{upcol}{fg=white, bg=red}       % Para la parte superior (header)
  \setbeamercolor{lowcol}{fg=black, bg=blue!20}    % Para la parte inferior (cuerpo)
Ejemplo de uso:
\begin{frame}{Ejemplo de recuadro personalizado}
    \begin{beamerboxesrounded}[upper=upcol, lower=lowcol, shadow=true]{Título del Bloque}
      Aquí va el contenido del recuadro.  
      \begin{itemize}
        \item Primer elemento
        \item Segundo elemento
      \end{itemize}
    \end{beamerboxesrounded}
  \end{frame}

Es importante tener en cuenta que estas opciones pueden depender del tema o inner theme que esté utilizando la presentación. Algunos temas pueden forzar estilos de bloque particulares, de modo que en ocasiones puede ser necesario ajustar o redefinir la plantilla de bloques para ver reflejados los cambios.

Esta estrategia permite usar diferentes combinaciones de colores para distintas cajas:

Definición de objetos de color personalizados:
% Definición de objetos de color personalizados
  \setbeamercolor{blueHeader}{fg=white, bg=Blue}
  \setbeamercolor{redBody}{fg=black, bg=Red!20}
  \setbeamercolor{greenHeader}{fg=white, bg=Green}
  \setbeamercolor{yellowBody}{fg=black, bg=Yellow!50}
Ejemplo de uso con diferentes colores:
\begin{frame}{Cajas de Diferentes Colores}
    \begin{beamerboxesrounded}[upper=blueHeader, lower=redBody, shadow=true]{Caja 1: Azul - Rojo}

Alertar o resaltar items en una lista:

Usar \alert para Resaltar Texto

Puedes usar el comando \alert{} para resaltar partes específicas del texto dentro de los items:

\begin{frame}
    \frametitle{Resaltar Texto con \alert}
    \begin{itemize}
      \item Este item contiene texto \alert{importante}.
      \item Este item contiene texto \alert{crucial}.
    \end{itemize}
  \end{frame}

Alertar Items con <alert@+>

Puedes utilizar el comando <alert@+> para resaltar los items de una lista uno a uno:

\begin{frame}
    \frametitle{Alertar Items}
    \begin{itemize}[<+-| alert@+>]
      \item Este item se muestra y resalta en la primera diapositiva.
      \item Este item se muestra y resalta en la segunda diapositiva.
      \item Este item se muestra y resalta en la tercera diapositiva.
    \end{itemize}
  \end{frame}

Resaltar Items con <alert@1>, <alert@2>, etc.

Puedes especificar qué item resaltar en cada diapositiva utilizando <alert@1>, <alert@2>, etc.:

\begin{frame}
    \frametitle{Resaltar Items Específicos}
    \begin{itemize}
      \item<alert@1> Este item se resalta en la primera diapositiva.
      \item<alert@2> Este item se resalta en la segunda diapositiva.
      \item<alert@3> Este item se resalta en la tercera diapositiva.
    \end{itemize}
  \end{frame}

4. Combinar Reveal y Alert

Puedes combinar el revelado de items con su resaltado:

\begin{frame}
    \frametitle{Combinar Reveal y Alert}
    \begin{itemize}[<+->]
      \item<1-| alert@1> Este item se muestra y resalta en la primera diapositiva.
      \item<2-| alert@2> Este item se muestra y resalta en la segunda diapositiva.
      \item<3-| alert@3> Este item se muestra y resalta en la tercera diapositiva.
    \end{itemize}
  \end{frame}

Personalizar el color de alerta

Para personalizar el color de alerta en Beamer, puedes utilizar el comando \setbeamercolor{alerted text}{fg=color} para cambiar el color del texto resaltado por \alert. Aquí tienes un ejemplo de cómo hacerlo:

Cambiar el Color de Alerta

\documentclass{beamer}

% Cambia el color de alerta a verde
\setbeamercolor{alerted text}{fg=green}

\begin{document}

\begin{frame}
  \frametitle{Alerta Personalizada}
  Este texto es \alert{importante}.
\end{frame}

\end{document}

En este ejemplo, el texto dentro de \alert{} se mostrará en verde.

Imágenes animadas

1. Convertir un Video en una Secuencia de Imágenes

Puedes utilizar herramientas como Photoshop, Filmora, VLC, o FFmpeg para convertir un video en una secuencia de imágenes. Por ejemplo, con FFmpeg puedes usar el siguiente comando:

ffmpeg -i input.mp4 output-%03d.png

Esto generará una serie de imágenes numeradas (output-001.png, output-002.png, etc.).

2. Crear una Animación en LaTeX Beamer

Una vez que tengas la secuencia de imágenes, puedes usar el paquete animate en LaTeX Beamer para crear una animación. Aquí tienes un ejemplo:

\documentclass{beamer}
\usepackage{animate}

\begin{document}

\begin{frame}
  \frametitle{Animación}
  \centering
  \animategraphics[autoplay,loop,width=\textwidth]{10}{output-}{001}{100}
\end{frame}

\end{document}

En este ejemplo, se reproduce una animación que muestra las imágenes output-001.png a output-100.png en un bucle.

Ventajas y Limitaciones

  • Ventajas: Esta técnica permite crear animaciones en PDFs sin necesidad de Flash.
  • Limitaciones: El tamaño del PDF puede aumentar significativamente si la animación tiene muchos frames. Además, no todos los visores de PDF pueden reproducir animaciones.

Herramientas para Convertir Videos a Imágenes

  • Photoshop: Puedes abrir un video en Photoshop y exportar cada frame como una imagen.
  • Filmora: Ofrece una forma sencilla de convertir videos a secuencias de imágenes.
  • VLC: Puedes usar VLC para extraer imágenes de un video.
  • FFmpeg: Una herramienta de línea de comandos poderosa para convertir videos a imágenes.
  • Blender: es Un software de modelado 3D pero su editor de video permite cargar un vídeo y renderizar fotogramas individuales

Con estas herramientas y técnicas, puedes crear animaciones en tus presentaciones Beamer utilizando secuencias de imágenes extraídas de videos.
