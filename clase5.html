Clase 5: Elementos avanzados y personalización

1. Inclusión de código fuente

LaTeX permite mostrar fragmentos de código con resaltado y numeración de líneas, especialmente útil para documentación técnica.

1.1 Paquete listings

El paquete listings facilita la inclusión de código fuente con resaltado de sintaxis:

\usepackage{listings}
\lstset{
    basicstyle=\small\ttfamily,
    keywordstyle=\color{blue}\bfseries,
    commentstyle=\color{gray},
    stringstyle=\color{red},
    numbers=left,
    frame=single,
    breaklines=true
}

1.2 Resaltado de sintaxis

\begin{lstlisting}[language=Python,caption={Ejemplo en Python},label={code:python}]
def factorial(n):
    if n <= 1:
        return 1
    else:
        return n * factorial(n - 1)
\end{lstlisting}
  • language=Python determina el lenguaje.
  • caption={...} añade un título descriptivo.
  • label={...} asigna una etiqueta para referencias en el documento.

2. Creación de entornos personalizados

Puedes definir tus propios entornos para dar un estilo específico a ciertos contenidos. Por ejemplo:

\newenvironment{miResumen}
{%
  \begin{quote}
  \bfseries Resumen:\quad
}
{%
  \end{quote}
}

Uso:

\begin{miResumen}
Este es el resumen de la sección.
\end{miResumen}

3. Diseño de página: márgenes, encabezados y pies de página

3.1 Márgenes con geometry

\usepackage[a4paper, margin=2cm]{geometry}

Ajusta la hoja A4 con un margen de 2 cm en cada lado.

3.2 Encabezados y pies de página con fancyhdr

\usepackage{fancyhdr}
\pagestyle{fancy}

\fancyhead{}    % Limpia encabezados
\fancyfoot{}    % Limpia pies de página

\fancyhead[L]{Nombre del Documento}
\fancyhead[R]{\leftmark}
\fancyfoot[C]{\thepage}
  • \leftmark mostrará el título de la sección o capítulo actual.
  • \thepage indica el número de página.

4. Uso de colores

Para usar colores, se carga el paquete xcolor o color:

\usepackage{xcolor}

4.1 Texto y fondo coloreados

\textcolor{blue}{Texto en azul.}
\colorbox{yellow}{Fondo amarillo para este texto.}

4.2 Definir colores personalizados

\definecolor{miverde}{RGB}{0,128,0}
\textcolor{miverde}{Texto en un verde personalizado.}

4.3 tcolorbox para recuadros de colores

tcolorbox permite crear recuadros con gran flexibilidad:

\usepackage{tcolorbox}

\begin{tcolorbox}[colback=blue!5!white,colframe=blue!75!black,title=Ejemplo de tcolorbox]
Este es un texto de ejemplo dentro de un recuadro de color personalizado.  
Puedes ajustar opacidad, bordes y mucho más.
\end{tcolorbox}

4.4 Parámetros de tcolorbox

En el ejemplo anterior:

Aparecen tres parámetros que configuran el aspecto y el título del recuadro:

  1. colback=blue!5!white
    • Indica el color de fondo (background) del recuadro.
    • blue!5!white significa que se mezcla el color azul con un 5% de intensidad y el resto con blanco (95%).
    • Este esquema produce un tono muy claro de azul.
  2. colframe=blue!75!black
    • Determina el color del borde (frame) del recuadro.
    • blue!75!black significa que se combina un 75% de azul con un 25% de negro, dando como resultado un azul oscuro.
  3. title=Ejemplo de tcolorbox
    • Muestra un título en la parte superior del recuadro.
    • El texto "Ejemplo de tcolorbox" aparece centrado sobre la caja.

Estos parámetros pueden personalizarse libremente para ajustar el color de fondo y de borde, así como etiquetar el recuadro con un título apropiado.

5. Ejemplo de colortbl en tablas

El paquete colortbl añade color a filas y celdas de tablas:

\usepackage{colortbl}
\usepackage{booktabs}

\begin{table}[ht]
\centering
\rowcolors{1}{blue!10}{white}
\begin{tabular}{l c c}
\toprule
\rowcolor{blue!30}
\textbf{Producto} & \textbf{Precio} & \textbf{Stock} \\
Manzanas  & \$2.00 & 50 \\
Peras     & \$1.80 & 30 \\
Naranjas  & \$2.20 & 40 \\
\bottomrule
\end{tabular}
\caption{Ejemplo de tabla con colortbl}
\label{tab:coloreada}
  • \rowcolors{1}{blue!10}{white} alterna el color de fondo en las filas, iniciando desde la fila 1.
  • \rowcolor{blue!30} ajusta el color de la fila del encabezado.
  • booktabs aporta comandos como \toprule, \midrule y \bottomrule.

6. Ejercicios prácticos

  1. Listados de código: Crea un fragmento con lstlisting donde incluyas un pequeño script en C, Python o cualquier otro lenguaje.
  2. Entornos personalizados: Define un entorno llamado aviso que muestre un recuadro con un texto resaltado.
  3. Diseño de página: Ajusta los márgenes a 3 cm y agrega tu nombre en el lado izquierdo del encabezado, el título de la clase en el lado derecho, y centra el número de página en el pie.
  4. Uso de colores:
    • Crea un tcolorbox de fondo gris con borde negro.
    • Añade una tabla con filas alternadas en color usando colortbl.
  5. Revisión de Git: Realiza algunos cambios en un archivo, pídele a un compañero de clase que haga cambios en el mismo archivo en una rama diferente, y luego practica la resolución de conflictos al hacer git pull.

7. Repaso de Git: resolución de conflictos simples

Cuando realizas git pull, si tu rama local y la remota tienen cambios en la misma línea, se genera un conflicto.

  1. Pull: 
    git pull
  2. Buscar y resolver el conflicto en el archivo marcado por: 
    <<<<< HEAD
...
=======
...
>>>>>> remote/branch
  3. Guardar y hacer commit: 
    git add archivo_conflictivo
git commit -m "Resolver conflicto"
  4. Push para actualizar el repositorio remoto: 
    git push

Este proceso enseña a manejar cambios simultáneos y fusionar esfuerzos en un equipo de trabajo.
