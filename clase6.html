Clase 6: Creación de una Plantilla para Tesis Doctoral en LaTeX

Clonar el repositorio

La plantilla completa se puede clonar con: git clone https://github.com/latex-git/plantilla-tesis.git

Paquetes y Configuración Inicial

El preámbulo del documento contiene una serie de paquetes que proporcionan funcionalidades específicas:

Paquetes Básicos

  • natbib: Gestiona citas bibliográficas.
  • inputenc y fontenc: Manejan la codificación de caracteres y fuentes.
  • babel: Adapta el documento al idioma español.
  • amsmath: Proporciona funcionalidades matemáticas avanzadas.

Formato y Diseño

  • geometry: Configura los márgenes y el tamaño del papel.
  • fancyhdr: Personaliza encabezados y pies de página.
  • graphicx: Permite la inserción de imágenes.
  • hyperref: Crea enlaces interactivos en el documento PDF.

Tablas y Figuras

  • longtable: Permite tablas que se extienden por varias páginas.
  • xcolor: Añade color a las tablas.
  • caption: Personaliza el formato de los pies de figura y tabla.

Estructura del Documento

La estructura de la tesis se organiza en secciones principales:

  1. Portada: Se crea utilizando comandos personalizados y el entorno center.
  2. Páginas Preliminares: Incluyen agradecimientos, dedicatoria y citas.
  3. Índices: Se generan automáticamente con \tableofcontents, \listoffigures, y \listoftables.
  4. Cuerpo Principal: Dividido en capítulos como Introducción, Material y Métodos, Resultados, etc.

Comandos Personalizados

El archivo define varios comandos personalizados para facilitar el formateo:

\newcommand{\figura}[4]{
  \begin{figure}[htbp]
    \begin{center}
      \includegraphics[width=#2\textwidth]{figuras/#1}
    \end{center>
    [#3]{#4}

Este comando simplifica la inserción de figuras, permitiendo especificar el nombre del archivo, tamaño, título corto y pie de figura en una sola línea.

Formato de Capítulos

El formato de los títulos de capítulos se personaliza utilizando \@makechapterhead:

\makeatletter
\def\@makechapterhead#1{
  \vspace*{1\p@}%
  {\parindent \z@ \raggedright \normalfont
    \ifnum \c@secnumdepth >\m@ne
      \par\nobreak
      \vskip 20\p@
    \fi
    \interlinepenalty\@M
    \Huge \bfseries \thechapter.\space#1\par\nobreak
    \vskip 40\p@
  }
}
\makeatother

Este código modifica la apariencia de los títulos de capítulos, ajustando espaciado y formato.

Configuración de Páginas

La plantilla utiliza fancyhdr para personalizar encabezados y pies de página:

\pagestyle{fancy}
\lhead{\uppercase{\bfseries{\leftmark}}}
\rhead{\uppercase{\bfseries{\rightmark}}}
\cfoot{\thepage}

Esto configura los encabezados con el nombre del capítulo y la sección actual, y coloca el número de página en el centro del pie.

Bibliografía

La bibliografía se configura para ser presentada en dos columnas y con un tamaño de fuente reducido:

\begin{singlespace}
\begin{footnotesize}
\begin{twocolumn}
\bibliography{biblio}
\end{twocolumn}
\end{footnotesize}
\end{singlespace}

Esta configuración optimiza el espacio y mejora la legibilidad de las referencias.

En conclusión, esta plantilla proporciona una base sólida para la creación de tesis doctorales en LaTeX, ofreciendo una estructura clara y herramientas para un formateo consistente y profesional.

Creación de las Páginas del "frontmatter"

El "frontmatter" en LaTeX contiene las páginas preliminares de la tesis, como portada, dedicatorias, agradecimientos e índices. Estas páginas suelen tener un formato distinto al cuerpo principal (por ejemplo, números de página en romanos y sin encabezados). A continuación, se explica cómo se construye cada una en la plantilla adjunta:

1. Portada Inicial

Objetivo: Mostrar título, autor, logos institucionales y fecha.
Código clave:

\begin{center}
        \LARGE \titulo \\ % Título definido con \newcommand
        \vspace{\stretch{2}} % Espacio flexible
        \includegraphics[width=.6\textwidth]{escudo_ugr_portada} % Logo
        \vspace{\stretch{2}} 
        \large Nombre Apellidos \\ % Autor
        Tesis Doctoral \\ % Grado
        \fecha % Fecha definida con \newcommand
    \end{center}
  • \vspace{\stretch{n}}: Distribuye el espacio verticalmente de forma proporcional.
  • \titulo y \fecha: Comandos personalizados definidos con \newcommand para reutilizar valores.

2. Página Institucional

Objetivo: Mostrar logos de instituciones y departamentos.
Técnica: Uso de minipage para alinear múltiples elementos en filas:

\begin{minipage}[t]{.23\textwidth}
        \centering
        \includegraphics[width=.8\textwidth]{logo_ib} % Logo 1
        Instituto de Biotecnología 
    \end{minipage}
    \hfill % Espacio horizontal
    \begin{minipage}[t]{.23\textwidth}
        \centering
        \includegraphics[width=.7\textwidth]{logo_ciencias} % Logo 2
        Facultad de Ciencias
    \end{minipage}

3. Página de Directores y Financiación

Objetivo: Detallar directores, programa de financiación y declaración institucional.

\begin{minipage}[t]{.4\textwidth}
        Dirigida por: \\ 
        \vspace{15ex} % Espacio fijo
        Dr. Fulanito de tal \\ % Director 1
        Dr. Menganito de cual % Director 2
    \end{minipage}

4. Dedicatoria y Cita

Objetivo: Añadir elementos personales o inspiradores.

\begin{flushright}
        \begin{minipage}{8cm}
            \textit{Siembra un pensamiento...} \\
            \begin{flushright} 
                Charles Reade
            \end{flushright>

5. Agradecimientos

Objetivo: Incluir un archivo externo con los agradecimientos.

\cleardoublepage % Asegura que empiece en página impar
    \input{agradecimientos.tex} % Archivo externo

6. Índices Generales

Objetivo: Listar capítulos, figuras y tablas.

\tableofcontents % Índice de capítulos
    \listoffigures   % Lista de figuras
    \listoftables    % Lista de tablas
    \clearpage       % Limpia la página
    \input{abreviaturas-utf.tex} % Lista de abreviaturas

7. Estilos de Página en "frontmatter"

  • \pagestyle{empty}: Elimina encabezados y pies de página.
  • \cleardoublepage: Fuerza una página nueva.
  • \thispagestyle{empty}: Aplica el estilo solo a la página actual.

Texto en columnas

Uso de la opción twocolumn en la clase de documento

Para generar un documento completo con formato de columnas, es posible utilizar la opción twocolumn al declarar la clase del documento. El siguiente ejemplo muestra cómo se aplica este formato a todo el documento:

\documentclass[twocolumn]{article}
\usepackage{graphicx}
\begin{document}

\section*{Documento en Dos Columnas}
Este documento se muestra en dos columnas de forma automática, lo que resulta útil para artículos, 
resúmenes o publicaciones científicas.
\includegraphics[width=\linewidth]{example-image}
Más texto de ejemplo se incluirá en ambas columnas, manteniendo un formato uniforme.

\end{document}

Uso del paquete multicol para columnas iguales

El paquete multicol permite dividir el contenido en varias columnas de igual ancho de forma sencilla. En el siguiente ejemplo se crea un entorno con dos columnas que contiene párrafos de texto y una lista:

\documentclass{article}
\usepackage{multicol}
\usepackage{lipsum} % Para texto de relleno
\begin{document}

\begin{multicols}{2}
\section*{Ejemplo de Dos Columnas}
\lipsum[1]

\begin{itemize}
    \item Elemento 1
    \item Elemento 2
    \item Elemento 3
\end{itemize}

\lipsum[2]
\end{multicols}

\end{document}

En este código, el entorno multicols divide automáticamente el contenido en dos columnas de igual ancho, permitiendo incluir secciones, párrafos y listas

Uso de entornos minipage para columnas de anchos distintos

Si se requiere disponer columnas de diferentes anchos o integrar elementos heterogéneos (como imágenes y tablas junto con texto), el entorno minipage resulta muy útil. En el siguiente ejemplo se crean dos columnas con anchos personalizados:

\documentclass{article}
\usepackage{graphicx}
\begin{document}

\noindent
\begin{minipage}[t]{0.4\linewidth}
\textbf{Columna 1: Texto}\\[5pt]
Este es un ejemplo de texto en la primera columna. Se pueden incluir párrafos, fórmulas y otros elementos.
\end{minipage}
\hfill
\begin{minipage}[t]{0.55\linewidth}
\textbf{Columna 2: Imagen y Tabla}\\[5pt]
\includegraphics[width=\linewidth]{example-image-a}
\vskip10pt
\begin{tabular}{|c|c|}
\hline
A & B \\
\hline
1 & 2 \\
\hline
\end{tabular}
\end{minipage}

\end{document}

Aquí se han definido dos minipage con anchos del 40% y 55% respectivamente, separando el contenido en función de las necesidades: en la primera columna se muestra texto y en la segunda, una imagen seguida de una tabla.

Integración de elementos mixtos en múltiples columnas

El entorno multicols también es adecuado para distribuir diferentes tipos de contenido en columnas múltiples. En el siguiente ejemplo se demuestra una disposición en tres columnas, donde cada una contiene elementos distintos: texto e imagen, una tabla y una lista:

\documentclass{article}
\usepackage{multicol}
\usepackage{graphicx}
\usepackage{lipsum}
\begin{document}

\begin{multicols}{3}
\section*{Columna 1: Texto e Imagen}
\lipsum[1]
\includegraphics[width=\linewidth]{example-image}
\columnbreak

\section*{Columna 2: Tabla}
\begin{tabular}{|c|c|}
\hline
\textbf{Item} & \textbf{Valor} \\
\hline
Uno & 10 \\
Dos & 20 \\
\hline
\end{tabular}

\columnbreak

\section*{Columna 3: Lista}
\begin{itemize}
    \item Elemento A
    \item Elemento B
    \item Elemento C
\end{itemize}
\end{multicols}

\end{document}

Cada columna se encarga de un tipo de contenido distinto y, gracias al entorno multicols, se consigue una distribución equilibrada en el espacio disponible.

Separación entre columnas

Espacio entre columnas

Se define mediante la longitud \columnsep. Por ejemplo, si se requiere aumentar el espacio entre columnas, se puede usar:

\setlength{\columnsep}{3em}

Este comando establece la distancia horizontal entre el contenido de cada columna.

Línea divisoria entre columnas

La presencia (y grosor) de una línea vertical se controla con la longitud \columnseprule. Por defecto, tiene un valor de 0pt (es decir, no aparece ninguna línea). Para hacer visible la separación, se puede configurar, por ejemplo:

\setlength{\columnseprule}{0.4pt}

Con este ajuste, LaTeX dibuja una línea de 0.4pt de grosor entre las columnas.

Consejos Clave

  1. Flexibilidad con \vspace{\stretch{n}}: Permite ajustar el espacio vertical sin valores fijos.
  2. Logos con includegraphics: Usa rutas relativas y ajusta el ancho con width=.7\textwidth.
  3. Mantenimiento Modular: Separa contenido en archivos .tex independientes.

Uso del paquete balance

Para equilibrar la última página puedes incluir el paquete en el preámbulo de tu documento:

\documentclass[twocolumn]{article}
    \usepackage{balance}
    \begin{document}
    % Contenido del documento

    % … texto, secciones, etc.

    % Coloca el comando \balance en algún punto de la última página,
    % preferiblemente en la columna izquierda:
    \balance

    % Contenido final, por ejemplo:
    \bibliographystyle{plain}
    \bibliography{miBibliografia}
    \end{document}

El comando \balance indica a LaTeX que a partir de ese punto se debe intentar repartir el contenido de la última página de manera equitativa entre ambas columnas. Es fundamental asegurarse de insertar el comando lo suficientemente temprano en la primera columna de la última página; de lo contrario, LaTeX podría ya haber fijado el contenido, y el balanceo no surtiría efecto.
